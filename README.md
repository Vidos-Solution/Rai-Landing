# Vidos Responsible AI (RAI) - Parent Website

This is the parent landing page for Vidos Responsible AI, serving as the central entry point that directs users to two specialized platforms:
- **For Professionals**: Learning platform (course.responsibleai.com)
- **For Businesses**: Governance framework platform (vidos.responsibleai.com)

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **React**: React 19
- **TypeScript**: TypeScript 5
- **Styling**: Tailwind CSS 4
- **Deployment**: Vercel (recommended)

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build for Production

```bash
npm run build
npm start
```
